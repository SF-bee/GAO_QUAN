# #2EVW0019 题解

## 解法 1：区间 dp

$dp_{i,j}$ 表示区间 $[i,j]$ 合法

1. 若 $A = ()$，$A$ 是合法的
2. 若 $A$ 是合法的，$(A)$ 是合法的
3. 若 $A,B$ 是合法的，$AB$ 是合法的
4. 合法的长度一定是偶数

对于第一个性质：可以预处理所有合法的最小括号串

对于第二个性质：可以得到：若 $dp_{i,j}$ 为真，当 $dp_{i + 1,j - 1}$ 为真且 $s_i,s_j$ 可以匹配

对于第三个性质：可以得到：若$dp_{i,j}$ 为真，当 $\exists k \in (i,j)$，$dp_{i,k},dp_{k + 1,j}$ 为真。

时间复杂度 $O(n^3)$ 期望得分 100pts，可以通过，但是不应该局限于此。

::::info[Code]

```cpp
#include <bits/stdc++.h>
const int MAX = 5e2;
template <typename T> using Row = std::array<T, MAX>;

std::string s;
std::array<Row<int>, MAX> dp;

int main() {
    std::cin.tie(0)->sync_with_stdio(0);

    std::cin >> s;

    for (int len = 2; len <= s.size(); len += 2) {
        for (int l = 0; l + len - 1 < s.size(); l++) {
            int r = l + len - 1;
            if (l == r - 1 && s[l] == '(' && s[r] == ')') {
                dp[l][r] = 1;
                continue;
            }
            for (int k = l; k < r; k++)
                if (dp[l][k] && dp[k + 1][r]) {
                    dp[l][r] = 1;
                    break;
                }
            if (dp[l + 1][r - 1] && s[l] == '(' && s[r] == ')')
                dp[l][r] = 1;
        }
    }

    int ans = 0;
    for (int len = 2; len <= s.size(); len++) {
        for (int l = 0; l + len - 1 < s.size(); l++) {
            int r = l + len - 1;
            if (dp[l][r])
                ans = std::max(ans, r - l + 1);
        }
    }
    std::cout << ans << '\n';
    return 0;
}
```

### 解法 2

设 $dp_{i}$ 表示以第 $i$ 个字符结尾的最长合法括号序列

- 对于第二个性质：若当前字符为 `(` 那么跳过。否则，我们可以查看它的前一个字符最远可以匹配多远，然后往前再数一个，看看能否匹配，如果可以那么就扩展。
