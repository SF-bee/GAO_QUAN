# 【第六周】阻拦乘积

## 题目描述

有 $n$ 座山排成一排，从左到右依次为第 $1\sim n$ 座山，第 $i$ 座山的高度为 $a_i$，Z 会尝试站在第 $x$ 座山上向右开炮，炮弹会在飞行过程中到达第一座高度大于等于 $a_x$ 的山或者飞过第 $n$ 座山之后被拦住，这次开炮的得分即为在被拦住之前经过的所有山（包括第 $x$ 座山）高度的乘积。  

由于 Z 是个笨蛋，他只想知道得分十进制下的最后四位组成的数（不包括前导 $0$，例如 $10043$ 的后四位组成的数为 $43$，$27102$ 的后四位组成的数字为 $7102$）。  

Z 有 $q$ 个开炮计划，每次他会告诉你他决定在哪座山上开炮，请你告诉他这次开炮得分的最后四位组成的数字是什么。

## 输入格式

第一行两个以空格隔开的整数 $n,q$，分别表示山的数量和开炮的次数。  
第二行有 $n$ 个以空格隔开的整数 $a_1,a_2,\cdots a_n$ 依次表示第 $1\sim n$ 座山的高度。  
接下来 $q$ 行，每行一个整数 $x_i$ 表示第 $i$ 次开炮所在的山的编号。

## 输出格式

$q$ 行，每行一个整数表示对应的那次开炮的得分最后四位组成的数字。

## 样例 #1

### 样例输入 #1

```
6 4
1 1 8 2 7 5 
5
2
3
1
```

### 样例输出 #1

```
35
1
560
1
```

## 提示

对于全部的测试数据满足：$1\le n,q\le 2\times 10^5,0\le a_i\le 10^9,1\le x_i\le n$。
